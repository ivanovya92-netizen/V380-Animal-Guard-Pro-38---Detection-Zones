# Pro33 Alarm Reminder Report

## Alarm behavior

When Smart Animal mode decides that siren is eligible, the app now:
1. Shows full-screen phone alarm.
2. Plays alarm ringtone if enabled.
3. Vibrates if enabled.
4. Shows persistent alarm notification.
5. Repeats reminders until ACK or max reminder count.
6. Supports Snooze.

## Protected parts

S01 direct siren constants are unchanged from Pro32.
CameraBoarVision stays enabled.
